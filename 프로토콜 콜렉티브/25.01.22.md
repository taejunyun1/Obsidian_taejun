> 정리된 키워드
- 1달 -> 중간 결과물  / 프로세스
- 한결
	- 규칙, 입력, 출력, 상전이, 해체 ,부식, 유기체, 무기체, 오류, 시네스지아(질감으로 변환), 역변환, 변환 거부, 일정한 규칙 생성 f(x), png<->jpg(샘플링), 기능, 소통, 알고리즘, 차원 간 소통,  확장, 지속성, 의미의 재배치, 반복 변환, 데이터 손실, 파괴 규칙 재구성
- 도영
	- 물질과 데이터 사이의 핑퐁게임
	- 현실과 데이터 사이의 물질을 만들어보고자함
	- 프란시스 알리시스, 얼음작업 차용
	- 크로마키나 페인트를 돌에 가득 묻히고 페인트를 벗겨가는 방식
	- 물질세계와 데이터의 사이에 있듯한 시각적인 느낌.
	- 통과시에 변화하는 물질의 여러 형태(용태)
		- **온전하지 못한 변환(피지컬-디지털)이라는 프로세스**
		- 노이즈, 원본과 복제 등으로 
		- 데이터가 구성되어 있거나 배치되어있는걸 다르게 구성하는것
		- 입력과 출력의 구조 / 유기체->무기체의 변환 / 규칙성-> 무규칙성
