
## 5. 한불수교 이후 실제로 들어온 것들

### 물질
- 서양식 인쇄기
- 카메라
- 광학 렌즈
- 지도 제작 도구
- 전기 장비
- 의학 기구

### 지식
- 해부학
- 천문학
- 측량학
- 화학
- 박물학적 분류 체계

### 이미지 체계
- 사진 초상
- 도시·지형 사진
- 박물 표본 이미지
- 지도와 도해


이때부터  
**세계는 글이 아니라 이미지와 수치로 설명되는 방식**이 조선에 도입된다.


기존 조선 사회:
- 자연을 직접 보고 체험
- 손으로 만지고 기록
- 회화적 재현

근대 이후:
- 렌즈로 세계를 봄
- 사진으로 고정
- 계측기로 수치화
- 표본으로 분류

→ **감각 → 데이터 → 이미지**라는 변환 체계가 작동하기 시작.


조선 전통 세계관
(도덕·유교·천하도)
        ↓ 충돌
서구 기술 세계
(렌즈·계측·통신·군함)
        ↓ 조약
국제법·통상·선교
        ↓ 유입
물질 + 지식 + 이미지
        ↓ 변화
감각 → 데이터 → 이미지 인간 탄생


작업동기
-  지원 목적 활동방향
- 뭔가 새로운 것을 제작하기보다는 현재의 작업 방향성에서 개념을 가지고 와서 작업을 제작하겠다.
- 활동  방향은 지금 작업의 주된 방향성 -> 데이터와  물성  그리고 이미지 이것을 한불수교 140주년의 역사적 베이스와 의의를 바탕으로 작업을 제작하려고 한다.
	- 물리적  이동 (배) -> 데이터의 이동 (데이터-케이블)
	- 고정되지 않은 현실 (회화적 이미지) -> 사진을 통해 고정된 현실 (사진적 이미지-고정된 이미지)
	- 수치화되지 않은 여러 것들 (정신-종교) -> 수치화된 현실과 지도, 유일신 (데이터화 된 지도와 종교)
	- 조선인은 물리적인 세계를 감각하는 아주 신체적인 인간
		- 그의 변화하는 어떠한 감각이라는 지점을 사진적 이미지로 제작
		- 그러나 이것이 조선인을 전면에 등장시킨다기보다는 그의 신체가 데이터화되즌 여러 감각을 이미지로 제작한다는 거임.

**외부 세계가 들어오는 장면**

### 이미지 소스

- 19세기 프랑스 군함 사진
    
- 강화도 해협 사진
    
- 초기 서양식 지도
    
- 바다 표면 매크로 촬영
    

### 제작 방식

- 여러 이미지를 레이어링
    
- 바다 이미지 위에 군함 실루엣을 반투명 합성
    
- 인쇄 후 재촬영(Rescan)

## 2. **Archive (기록)**

**새로운 세계가 축적되는 과정**

### 이미지 소스

- 초기 조선인 초상사진
    
- 서양식 병원 기록 사진
    
- 지도, 해부도, 식물 표본 이미지

- 표본 이미지와 신체 이미지의 접합
