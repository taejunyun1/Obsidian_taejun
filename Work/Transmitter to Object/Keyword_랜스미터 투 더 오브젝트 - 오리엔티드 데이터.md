## Process
- 유기적신체(아날로그) + 정밀한 스크리(Pixel의 세계)
- 유기적신체임과 동시에 디지털 + 디지털세게의 높은 해상도
	- 클릭, 드래그, 타이핑, 바라보기 등 일랸의 행동(action)
- 위의 혼합된 과정은 데이터Data -> <-오브젝트의 통신과정을 치환된
	- 이 과정에서는 결국 왜곡, 전송, 변화, 오류가 일어나게됨
	- **이러한 과정을 작업의 시각적인 형태로 변환하게되는 것임.**
- 전기 (아날로그적인 통신방법) - 아날로그적인 신체에서 통신하여 무언가를 지각하게됨
	- 이러한 전기는 물리적 입력을 통해 물리적인 오브젝트르 감각함.
	- 변형된 사물
		- 왜곡되거나 일그러진 사물
	- 설치+사진
		- 여러매체의 결합
		- 빛 LED 무빙이미지
	- 사진
	- 사운드+이미지
- 이러한 형태로 변환가능함.

**“노동하는 신체”에서 “새로운 형태의 노동”으로의 전환**

기존의 노동은 육체적 행위를 기반으로 했지만, 디지털 시대에는 신체의 미세한 움직임(예: 눈의 이동, 클릭, 드래그, 팔로우, 타이핑 등)이 새로운 노동의 형태로 자리 잡고 있음.

1.미시적 신체 움직임

•눈의 움직임(시선 추적) → 광고 타겟팅, 데이터 수집
•마우스 클릭 → 알고리즘 피드백, 사용자 경험 분석
•드래그 → 디지털 디자인, UX 조작
•팔로우 → SNS 알고리즘 강화, 영향력의 노동
•타이핑 → 생성 AI, 데이터 축적
