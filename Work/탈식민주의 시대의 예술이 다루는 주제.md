			- 노동의 형태를 어떻게 보여줄 것인가?
				- 현대적인 노통의 형태를 작업의 형태로 재현하는 것에 목적을 두고 작업을 진행함.
				- 두가지로 구분
				>	전통적-산업적인 노동의 형태
				>	- **신체적 성격 -곡물 수확 기게조작 부품조립등의 단순 반복동작** (근육의 수축과 이완을 통한 노동의 형태) - > 신체적 필로와 감각에 대한 노동의 형태 감각위주
				>	- **디지털 사무 노동 키보드 타이핑 마우스 조작 화명응시 등의 주로 손과 눈을 활용한 정적 반복동장** (데이터입력 / 파일정리 / 문서검토) -> 신체적 긴장에 대한 감각 위주

			 - ==모스부호 ==- 힘의 압력을 전기자기적인 신호로 변환해 데이터로 전달
			 - 소나 Sonar - 음파탐지기 음파를 보내 돌아오는 방향성과 데이터를 분석하여 시각적인 대상으로 재현

			 - 다우징 로드 Dawzing Rod - 다우징로드를 통해 눈에 보이지않는 대상을 드러내는 방식으로 사용

			 - 지진계 Earthquake Transmitter - 지진계는 지각의 미세한 떨림과 지진파를 감지하고 기록하는 장비 -> 이걸 데이터의 전송과 전자기의 이동을 기록하도록 변환해보면 어떻까?
